## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke trust and security.
- Background color: Light Gray (#EEEEEE) for a clean and professional look.
- Accent color: Amber (#FFC107) to highlight important information and CTAs.
- Body and headline font: 'Inter', sans-serif, for a clean and readable experience.
- Use clear, consistent icons from a library like Material Icons to represent transaction categories and other actions.
- Implement a responsive, card-based layout to organize information effectively across devices.
- Use subtle animations and transitions to provide feedback on user actions and guide navigation.
